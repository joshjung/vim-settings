# vim-settings
Github with my VIM rc and various scripts to setup vim.

# Installation

Clone to your system and then run `./install.sh`. You will be prompted before installation takes place.

# Included Settings

- No wrap
- Line numbers
- Pathogen
- Sensible
- Solarized theme (dark)
- NERDTree
